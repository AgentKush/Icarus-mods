## Installation
1. Download the `.EXMODZ` file
2. Import into JimK72's Mod Manager
3. Enable the mod and launch the game
4. Find the weapons in the Workshop under a new BioLab category

## Compatibility
- Works with Icarus base game and all DLCs
- Compatible with FastProcessing mods
- Compatible with Workshop_Recyclers mod
