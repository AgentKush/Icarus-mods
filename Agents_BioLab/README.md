# Agent's BioLab

**Version:** 1.7  
**Author:** AgentKush  
**Compatible with:** JimK72's Icarus Mod Manager (EXMODZ format)

## Description

Adds all Bio Lab Legendary weapons to the Workshop! Purchase these powerful weapons using Exotics from a dedicated workshop tab.

## Features

- New Workshop tab: **Agent's BioLab**
- 10 legendary weapons in a 2x5 grid layout
- All weapons cost **1,000 Exotics** each (Research + Craft)
- Weapons arranged in neat rows for easy navigation

## Installation

1. Download `Agents_BioLab.EXMOD` (or `.EXMODZ` if zipped)
2. Place in your Mod Manager's `Extracted_Mods` folder
3. Use JimK72's Mod Manager to merge and install
4. Launch Icarus and check the Workshop for "Agent's BioLab" tab!
