# Agent's Items - Complete Workshop

**Version:** 4.9  
**Author:** AgentKush  
**Compatible with:** JimK72's Icarus Mod Manager (EXMODZ format)

## Description

A comprehensive addition to the Workshop with **1,500+ items** available for purchase! This mod adds nearly every item in the game to a dedicated workshop tab.

## Features

- New Workshop tab: **Agents Items**
- **1,500+ items** organized and purchasable
- Vestiges included
- Organized armor sets by tier
- All resources and materials available
- Tiered pricing system using Credits

## Pricing System

Items are priced based on their tier and complexity:

| Tier | Research Cost | Craft Cost |
|------|---------------|------------|
| Basic (wood, stone, fiber) | 10 Credits | 5 Credits |
| Common (iron, copper, bone) | 25 Credits | 15 Credits |
| Uncommon (steel, gold) | 50 Credits | 25 Credits |
| Rare (titanium, platinum) | 100 Credits | 50 Credits |
| Exotic | 200 Credits | 100 Credits |

## Item Categories

- Tools (all tiers)
- Weapons (melee, ranged, firearms)
- Armor sets (complete sets from Cloth to Meta)
- Building materials
- Crafting materials
- Resources
- Seeds and farming items
- Decorations and furniture
- Vestiges (rare collectibles)

## Installation

1. Download `Agents_Items.EXMOD` (or `.EXMODZ` if zipped)
2. Place in your Mod Manager's `Extracted_Mods` folder
3. Use JimK72's Mod Manager to merge and install
4. Launch Icarus and check the Workshop for "Agents Items" tab!
