# Agent's Individual Item Kits

**Author:** AgentKush  
**Version:** 2.0

## Description
1396 individual item kits available in the Workshop! Each kit contains a single item, allowing you to purchase exactly what you need. Includes craftable items and raw materials, all fairly priced by complexity and rarity.

## Features
- **1396 individual items** available for purchase
- **73 organized categories** with visual spacing for easy browsing
- **Fair pricing system** based on item complexity (5-300 credits)
- **Search friendly** - find any item quickly

## Pricing Tiers
| Tier | Price | Item Types |
|------|-------|------------|
| Basic | 5-20 | Raw materials, basic resources |
| Standard | 25-50 | Processed materials, simple tools |
| Advanced | 75-150 | Complex items, equipment |
| Premium | 200-300 | Rare items, high-tier gear |

## Installation
1. Download the `.EXMODZ` file
2. Import into JimK72's Mod Manager
3. Enable the mod and launch the game
4. Browse the Workshop to find individual items by category
