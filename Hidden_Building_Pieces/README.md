# Hidden Building Pieces

**Author:** AgentKush  
**Version:** 1.0.7

## Description
Unlocks 18 hidden diagonal and curved building piece sets for Concrete, Glass, and Clay Brick. These pieces exist in the game files but are normally inaccessible. Now you can craft them at various benches!

## Building Sets Included

### Concrete (6 sets)
- Concrete Diagonal Set
- Concrete Diagonal Curved Set
- Concrete Wall Curved
- Concrete Diagonal Curved Wall Angles
- Concrete Roof Curved Angles
- Concrete Advanced Beam Set

### Glass (6 sets)
- Glass Diagonal Set
- Glass Diagonal Curved Set
- Glass Wall Curved
- Glass Diagonal Curved Wall Angles
- Glass Roof Curved Angles
- Glass Advanced Beam Set

### Clay Brick (6 sets)
- Clay Brick Diagonal Set
- Clay Brick Diagonal Curved Set
- Clay Brick Wall Curved
- Clay Brick Diagonal Curved Wall Angles
- Clay Brick Roof Curved Angles
- Clay Brick Advanced Beam Set

## Crafting Locations
- Masonry Bench
- Cement Mixer
- Electric Masonry Bench
- Glassworking Bench

## Installation
1. Download the `.EXMODZ` file
2. Import into JimK72's Mod Manager
3. Enable the mod and launch the game
4. Craft the new building pieces at the appropriate benches

## Compatibility
- Works with Icarus base game and all DLCs
- **Fully compatible** with FastProcessing mods (v4.0+)
- Compatible with all other AgentKush mods
