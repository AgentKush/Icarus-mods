# Hidden Building Pieces

![Version](https://img.shields.io/badge/version-1.0.7-blue)

## Overview

Unlocks 18 hidden building piece sets that exist within the game files but remain inaccessible in vanilla gameplay. Adds diagonal and curved architectural elements for Concrete, Glass, and Clay Brick materials.

## Features

### Concrete Building Sets
| Set Name | Pieces Included |
|----------|-----------------|
| Diagonal Set | Angled floor/wall pieces |
| Diagonal Curved Set | Curved diagonal elements |
| Wall Curved | Curved wall sections |
| Diagonal Curved Wall Angles | Corner transitions |
| Roof Curved Angles | Curved roofing pieces |
| Advanced Beam Set | Structural beam variants |

### Glass Building Sets
| Set Name | Pieces Included |
|----------|-----------------|
| Diagonal Set | Angled glass panels |
| Diagonal Curved Set | Curved glass elements |
| Wall Curved | Curved glass walls |
| Diagonal Curved Wall Angles | Glass corner pieces |
| Roof Curved Angles | Curved glass roofing |
| Advanced Beam Set | Glass beam variants |

### Clay Brick Building Sets

## Installation

1. Download the `.EXMODZ` file from this repository
2. Open JimK72's Icarus Mod Manager
3. Import the mod file
4. Enable and launch Icarus
5. Craft new pieces at the appropriate workstation
