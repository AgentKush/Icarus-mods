> **Unlock 18 hidden diagonal and curved building piece sets for Concrete, Glass, and Clay Brick!**

These building pieces exist in the game files but aren't normally available through crafting. This mod makes them craftable at the appropriate workstations.

---

## üì¶ What's Included

### Materials (3)
- **Concrete** - Craft at Masonry Bench, Cement Mixer, or Electric Masonry Bench
- **Glass** - Craft at Glassworking Bench
- **Clay Brick** - Craft at Masonry Bench or Electric Masonry Bench

### Building Sets Per Material (6)
| Set | Description |
|:----|:------------|
| Diagonal Set | Cross-tile diagonal building pieces |
| Diagonal Curved Set | Cross-tile curved pieces |
| Wall/Roof Curved | Curved wall versions for use under curved roofs |
| Curved Wall Angles | Cross-tile curved walls in various angles |
| Roof Curved Angles | Curved roof pieces for over/under curved wall angles |
| Advanced Beam Set | Advanced beams for diagonal and curved pieces |

**Total: 18 craftable item sets**

## üî® Crafting Recipes

### Concrete Pieces
| Recipe | Workstation | Materials |
|:-------|:------------|:----------|
| Diagonal/Curved Sets | Masonry Bench, Cement Mixer, Electric Masonry | 20 Concrete Mix + 2 Steel Rebar |
| Advanced Beam Set | Masonry Bench, Cement Mixer, Electric Masonry | 10 Concrete Mix + 1 Steel Rebar |

### Glass Pieces
| Recipe | Workstation | Materials |
|:-------|:------------|:----------|
| Diagonal/Curved Sets | Glassworking Bench | 2 Steel Ingot + 20 Glass + 6 Epoxy |
| Advanced Beam Set | Glassworking Bench | 1 Steel Ingot + 10 Glass + 3 Epoxy |

### Clay Brick Pieces
| Recipe | Workstation | Materials |
|:-------|:------------|:----------|
| Diagonal/Curved Sets | Masonry Bench, Electric Masonry | 20 Clay Brick + 10 Stone |
| Advanced Beam Set | Masonry Bench, Electric Masonry | 10 Clay Brick + 5 Stone |

## üìù Installation

### Using JimK72's Mod Manager (Recommended)
1. Add this repository URL to the Mod Manager's modinfo sources
2. Find "Hidden Building Pieces" and install
3. Click Merge to apply

### Manual Installation
1. Download `Hidden_Building_Pieces.EXMODZ`
2. Place in: `Icarus_Mod_Manager_2_x_x/Extracted_Mods/`
3. Use the Mod Manager to merge and install
