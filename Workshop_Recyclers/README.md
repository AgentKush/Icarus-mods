## Overview

Introduces two dedicated recycling machines that enable material recovery from both Workshop purchases and crafted items. Features 1,707 total recycling recipes across both systems.

## Features

### Biofuel Recycler

| Specification | Value |
|---------------|-------|
| Function | Converts Workshop items to Biofuel currency |
| Recipes | 309 |
| Use Case | Recover value from unwanted purchases |

### Organic Residue Recycler

| Specification | Value |
|---------------|-------|

## Installation

1. Download the `.EXMODZ` file from this repository
2. Open JimK72's Icarus Mod Manager
3. Import the mod file
4. Enable and launch Icarus
5. Craft recyclers at the appropriate workstation

## Gameplay Notes

- Partial value recovery on recycled items
- Enables inventory cleanup without total loss
- Strategic for managing Workshop credit economy
- Useful for recovering materials from outdated gear
