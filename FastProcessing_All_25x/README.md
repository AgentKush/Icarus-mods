# FastProcessing_All_25x

**Author:** AgentKush  
**Version:** 3.0

## Description
All crafting and processing 25x faster! Contains 1723 recipes including cooking, smelting, crafting, Fabricator, Material Processor, and more.

## Installation
1. Download the `.EXMODZ` file
2. Place it in your Icarus mods folder or use JimK72's Mod Manager
3. Enable the mod and launch the game
